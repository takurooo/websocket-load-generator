# websocket-load-generator


# Usage
```
Usage of clients.go
  -a string
    	http service address (default "ws://localhost:8000/")
  -c int
    	connections (default 1)
  -d int
    	duration (default 1)
  -l int
    	message length (default 1)
  -u string
    	duration unit(msec or sec) (default "sec")
```
